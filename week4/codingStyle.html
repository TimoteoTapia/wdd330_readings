These are the following points to take into account for coding style

  • Curly Braces

    • it need a space before the opening bracket and be in the same line of the function statement

  • Line Length

    • it is helpful to split the text, usually between 80 to 120 characters

  • Indents

    • the idea is to leave space before to star a statement

  • Semicolons

    • it is place after each statement (;)

  • Nesting Levels

    • in this case the best practice is to avoid nesting code

  • Function Placement

    • Code first, then functions
