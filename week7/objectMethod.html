Object Methods, "This"

In object-oriented programming, objects are used to represent real world entities. Objects can have properties and perform actions, which are represented by functions called methods. Methods are functions that are assigned as properties of an object.
The "this" keyword in JavaScript is used inside methods to refer to the object on which the method is being called. It allows methods to access and manipulate the object's properties. When a method is called using the object.method() syntax, the value of "this" inside the method refers to the object itself. 
             
              // Define an object with properties and methods
              let user = {
                firstName: "Luis",
                lastName: "Alberto",
                fullName: function() {
                  return this.firstName + " " + this.lastName;
                },
                greet: function() {
                  alert("Hello, " + this.fullName() + "!");
                }
              };
        
              // Call the methods of the user object
              function sayHello() {
                user.greet();
              }

CSS Animations

  1. CSS Animations:

    CSS animations allow simple animations to be created without using JavaScript.

  2. CSS Transitions:

    CSS transitions are used to animate changes in CSS properties. When a property changes, the browser automatically animates the transition based on the specified duration and timing function. Transition properties include transition-property, transition-duration, transition-timing-function, and transition-delay.

  3. Transition Properties:

    • transition-property specifies the properties to be animated.
    • transition-duration determines the duration of the transition.
    • transition-delay sets a delay before the transition starts.

  4. Transition Timing Function:

    The timing function describes how the animation progresses over time.

  5. Steps:

    The steps() timing function allows the transition to be divided into discrete steps.

  6. Transition End Event:

    The transitionend event is triggered when a CSS transition finishes.

             
            <\button id="animateBtn">Animate<\/button>
            <\div id="box"><\/div>
                #box {
                    width: 100px;
                    height: 100px;
                    background-color: blue;
                    transition-property: background-color, transform;
                    transition-duration: 1s;
                }
                //it will animate when animate class exist
                #box.animate {
                    background-color: red;
                    transform: translateX(200px) rotate(180deg);
                }
            
                //Code that I am using
                const animateBtn = document.getElementById('animateBtn');
                const box = document.getElementById('box');
                
                //add the new class "animate" and trigger the animation
                animateBtn.addEventListener('click', function () {
                    box.classList.add('animate');
                });

                //remove the class allowing repeat the animation
                box.addEventListener('transitionend', function () {
                    box.classList.remove('animate');
                });
